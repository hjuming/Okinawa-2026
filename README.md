# 沖繩 2026 旅遊手冊

靜態網頁：2026 沖繩親子冒險旅遊手冊。

## 本機預覽

以瀏覽器開啟 `index.html`，或使用本機伺服器：

```bash
npx --yes serve .
```

## GitHub Pages

1. 在 GitHub 儲存庫 **Settings → Pages**
2. **Build and deployment**：Source 選 **Deploy from a branch**
3. Branch 選 **main**，資料夾 **/ (root)**，儲存

完成後網址約為：`https://hjuming.github.io/Okinawa-2026/`
